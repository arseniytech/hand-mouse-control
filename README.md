# Hand Mouse Control

Control your mouse using your hand and camera gestures!  
- Move the mouse: index finger
- Click: pinch thumb and index finger

## Installation

```bash
git clone https://github.com/arseniytech/hand-mouse-control
cd hand-mouse-control
pip install -r requirements.txt

```
## Usage 

```bash
python hand_mouse.py
```
Move your index finger to control the mouse
Pinch your thumb and index finger to click
Press q to quit the program
